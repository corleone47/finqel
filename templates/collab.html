Contribute by adding other sites or improving the current one

There are many other sites that many would find relevant to track (Brazzers, Tushy, Bangros, RealityKings, LetsDoeIt and so forth) You can contribute by pulling the github repo and doing likewise with any other site you think is worth the hassle. The whole fetching should be automated, just like this is, so that no manual intervention is needed to keep track of relevant data. You can also improve the style and layout or any other design question you consider could be added to better the look and functionality of the current page.

Visit GitHub Repo

How does this differ from current sites that compile similar data?

In a nutshell, this site aims to show what's in the target page right now and, if available, when the next update is coming. Current sites that do this (e.g.: assoholics) compile recent updates but don't usually show the latest update of the site nor when the next update is coming.
